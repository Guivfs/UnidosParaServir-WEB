Política de Privacidade e Proteção de Dados

1. Introdução

A plataforma UPS - Unidos para Servir está comprometida com a proteção da privacidade e dos dados pessoais de seus usuários, conforme estabelecido pela Lei nº 13.709/2018 - Lei Geral de Proteção de Dados Pessoais (LGPD). Esta política descreve como coletamos, usamos, armazenamos e protegemos os dados fornecidos pelos usuários em nossa plataforma.

2. Coleta de Dados

Nós coletamos os seguintes dados pessoais:

  • Nome completo
  • Endereço de e-mail
  • Telefone
  • Informações sobre experiências e preferências de voluntariado
  • Dados de localização, quando autorizados pelo usuário

Finalidade da Coleta: Todos os dados coletados são necessários para facilitar o processo de recrutamento para atividades de voluntariado, permitir a comunicação entre as partes interessadas e personalizar as oportunidades oferecidas.

3. Uso e Tratamento dos Dados

Os dados pessoais fornecidos serão utilizados para:

  • Cadastro e autenticação do usuário;
  • Comunicação sobre oportunidades de voluntariado e atualizações da plataforma;
  • Personalização das ofertas de voluntariado com base nos interesses e localização;
  • Melhorar a experiência do usuário no site.

4. Compartilhamento de Dados

Os dados pessoais coletados podem ser compartilhados com:

  • Organizações parceiras que oferecem oportunidades de voluntariado;
  • Autoridades legais, quando requisitado por lei, para cumprimento de obrigações legais.
